# مذاكر

نموذج واجهي لتطبيق يساعد الطالب على رفع المنهج الدراسي وتحويله إلى بنك أسئلة ومسابقات مقسمة حسب الفصول.

## التشغيل

افتح `index.html` مباشرة في المتصفح. التطبيق يعمل محليًا بدون خادم.

## ما الذي يفعله النموذج؟

- يقرأ ملفات PDF أو ملفات نصية أو محتوى ملصقًا.
- يقرأ ملفات بنك أسئلة JSON جاهزة ويعرضها مباشرة كفصول ومسابقات.
- يعرض جودة النص المستخرج ومعاينة مباشرة قبل/بعد التوليد.
- يحاول تشغيل OCR تلقائيًا عند ضعف النص المستخرج من PDF.
- يقسم المنهج إلى فصول اعتمادًا على عناوين مثل: الفصل، الوحدة، الباب، Chapter، Unit.
- يولد أسئلة اختيار من متعدد، صح وخطأ، وبطاقات مراجعة.
- يتيح بدء مسابقة حسب فصل محدد أو كل الفصول.
- يصدّر بنك الأسئلة بصيغة JSON.

## ملفات المنهج المحولة

تم إنشاء ملفات جاهزة في `generated-curriculum/`:

- `math3-clean-question-bank.json`: الملف الأفضل للرفع داخل التطبيق.
- `chapter-01-clean-questions.json`: أسئلة الفصل الأول.
- `chapter-02-clean-questions.json`: أسئلة الفصل الثاني.
- `chapter-03-clean-questions.json`: أسئلة الفصل الثالث.
- `math3-readable-clean.md`: نسخة مقروءة ومنظمة للفصول والدروس.

تم نسخ الملفات الجاهزة أيضًا إلى `data/` داخل التطبيق، ويمكن تحميل الملف الكامل من زر `تحميل منهج الرياضيات` في شاشة الرفع.

## ملاحظة

هذا نموذج أولي يعمل داخل المتصفح. قراءة PDF وOCR تحتاج اتصال إنترنت لتحميل PDF.js وTesseract.js. عند تشغيل OCR على ملفات كبيرة جدًا، يعالج التطبيق أول 40 صفحة لتجنب بطء المتصفح؛ يمكن تقسيم الملف أو استخدام PDF نصي كامل. دعم Word والتحليل الذكي المتقدم يحتاج طبقة خلفية أو تكاملًا مع خدمة ذكاء اصطناعي.
